Introduction to HTML Tables

Tables in HTML are created using the <table> element. Tables are helpful when you need to display data in a structured format, like in reports, lists, or schedules.

  • <table>: Defines the entire table structure.
  • <tr>: Defines a row within the table.
  • <th>: Defines a header cell within a row.
  • <td>: Defines a standard data cell.

In this tutorial, we will walk through building a table from scratch, with detailed explanations of each component.

Step 1: The Basic Table Structure

To create a basic table, you only need to use the <table> element.

  </table>
Step 2: Adding Rows

To add rows to the table, use the <tr> tag. Each row represents a single horizontal line of data.

Example:

<table>
 <tr>
  <td>Row 1, Column 1</td>
  <td>Row 1, Column 2</td>
 </tr>
</table>

Output:

Row 1, Column 1 Row 1, Column 2
Step 3: Adding Columns

Columns are added within the rows using the <td> tag. Each <td> element represents a data cell in the row.

For example, we can create a table with multiple columns:

<table>
 <tr>
  <td>Column 1</td>
  <td>Column 2</td>
 </tr>
 <tr>
  <td>Row 2, Column 1</td>
  <td>Row 2, Column 2</td>
 </tr>
</table>

Output:

Column 1 Column 2
Row 2, Column 1 Row 2, Column 2
Step 4: Adding Table Headers

Table headers are defined using the <th> element. These headers are bold and centered by default, making them stand out as titles for each column.

Example:

<table>
 <thead>
  <tr>
   <th>Header 1</th>
   <th>Header 2</th>
  </tr>
 </thead>
 <tbody>
  <tr>
   <td>Row 1, Column 1</td>
   <td>Row 1, Column 2</td>
  </tr>
 </tbody>
</table>

Output:

Header 1 Header 2
Row 1, Column 1 Row 1, Column 2
Step 5: Styling the Table

Now that we have the table structure, let's make it look better with some simple CSS styles. Here's an example of how to apply a basic border and alternate row colors:

<style>
 .table {
   border-collapse: collapse;
   width: 100%;
   border: 1px solid #ddd;
 }
</style>
